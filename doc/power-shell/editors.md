## vi command mode `:`

- `:wq!` save and quit
- `:r file` read file
- `:w (file)` write file
- `q!` quit without saving
- `:w !sudo tee %` write as user

## vi input mode

- `a` insert char right at the cursor
- `i` insert char left to the cursor
- `o` add a new line after the current line
