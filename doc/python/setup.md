## setup on debian linux

```sh

# install python
sudo apt install python3

# check version
python3 --version

# install package manager
sudo apt install python3-pip

# check pip version
pip3 --version

```

## setup on alpine linux

```sh

apk add --no-cache python3 py3-pip

```
